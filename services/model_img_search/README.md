# Сервис для поиска по изображению и тексту с помощью моделей глубокого обучения
Данный сервис реализует функционал поиска по изображению или тексту наиболее 
соответствующих изображений из имеющихся в БД. Для поиска соответствующих 
записей в БД использовались hnsw-индексы на основе метода вычисления косинусной 
меры векторной базы данных pgvector. 
Использует следующие исполняемые файлы:
- `main.py`: реализует функционал обращение к методам поиска соответствующих 
    запросу изображений через API;
- `resnet_img_search.py`: в этом файле реализован метод поиска изображений, 
    наиболее соответствующих переданному в запросе изображению, с помощью
    предобученной модели классификатора ResNet34;
- `clip_img_search.py`: в этом файле реализован метод поиска изображений, 
    наиболее соответствующих переданному в запросе тексту, с помощью
    предобученной модели CLIP;
- `grid_plot.py`: в этом файле реализован метод склейки в коллаж-сетку всех 
    найденных в вышеописанных алгоритмах изображений в одно большое изображение
    для возврата пользователю в качестве ответа на его запрос.
